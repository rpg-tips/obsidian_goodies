# obsidian_goodies

Templates, random tables, character sheets and other stuff for playing solo RPGs with Obsidian.

## Recommended plugins:

- Dice Roller.
- Initiative Tracker.
- TTRPG Statblocks.
- Admonition.
- Leaflet.
- Advanced Tables.
- Folder Note.
- Folder Icons.
- Templater.
- Dataview.

## Nice plugins but not essential

- Banners.
- Supercharged Links.
- Collapse All.
- Homepage.
- Timelines.
- Tag Wrangler.
- Gallery.
